# GOоseTracker

## Описание

**GOoseTracker** — браузерное расширение для легкого и быстрого взаимодействия руководителя и участников его команды для закрытия небольших рабочих вопросов.

Расширение позволяет оперативно обмениваться сообщениями и уведомлениями между сотрудниками и руководителем, сокращая время на решение повседневных рабочих задач.

---

# Запуск Backend

## Требования

Перед запуском убедитесь, что установлены:

- Go (версия, указанная в `go.mod`)
- Docker
- Docker Compose
- Make

Также необходимо создать файл `.env` с настройками подключения к базе данных.

---

## 1. Запуск PostgreSQL

```bash
make db-up
```

---

## 2. Применение миграций

```bash
make migrate-up
```

---

## 3. Запуск приложения

```bash
make run
```

После запуска API будет доступно по адресу:

```
http://localhost:8080
```

Swagger:

```
http://localhost:8080/swagger/index.html
```

---

# Работа с миграциями

### Создание новой миграции

```bash
make migrate-create seq=<migration_name>
```

### Применить миграции

```bash
make migrate-up
```

### Откатить последнюю миграцию

```bash
make migrate-down
```

# Подключение браузерного расширения

1. Откройте Google Chrome.
2. Перейдите по адресу:

```
chrome://extensions
```

3. Включите **Режим разработчика (Developer mode)**.
4. Нажмите **Загрузить распакованное расширение (Load unpacked)**.
5. Выберите папку, содержащую файл **`manifest.json`**.
6. После загрузки расширение появится в списке установленных и будет готово к использованию.

# Структура проекта

```
.
├── backend/
│   ├── cmd/                    # Точка входа приложения
│   ├── internal/               # Бизнес-логика приложения
│   ├── docs/                   # Swagger-документация
│   ├── migrations/             # SQL-миграции
│   ├── docker-compose.yml      # Конфигурация Docker Compose
│   ├── Makefile                # Команды для запуска и миграций
│   └── .env                    # Переменные окружения
│
└── frontend/
│   ├── AdminPanel/             # Админ панель
│   ├── Duck/                   # Работа уведомления-гуся
│   ├── LogIn/                  # Регистрация и вход для руководителя и учасников его команды
│   └── manifest.json           # Манифест расширения Google Chrome
```
